1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

### Traditional Hosting
Upload all files including:
- index.html
- style.css
- script.js
- send-email.php
- fetch-oegussa-prices.php

## Configuration
Update email in `send-email.php` line 3:
```php
$to_email = "your-email@example.com";
```
